Version 0.2.60210.2

This is a simple "to do" application with basic functionality.
It uses HTML (some HTML5 features), JavaScript, jQuery library, and just a little CSS.

    Features:
  • Add new task.
  • Name limitation - up to 32 characters.
  • Edit task's name.
  • Remove a task, when it is done.
  • Represent every task as an object with own propreties.
  • All task objects are stored in an array.
  • Task list is represented on the web page.
  • Sequense of div-elements (with sub-elements) is used, instead of ordered/unordered HTML lists.
  • A simple Status pannel.
  • Show/Hide different pannels.
