To-Do App

This is a simple "to-do" application with basic functionality. It uses HTML5, CSS, W3.CSS, JavaScript and jQuery.


Github repository:
https://github.com/atorov/todoapp

    Features:

  • Add new task
  • Edit task's name
  • Remove a task, when it is done
  • A simple Status pannel
  • Show/Hide different pannels

    Hints:

  • Click on pannel title to show/hide
  • Continuous random clicking here for more than 24 hours may cause brain damage
