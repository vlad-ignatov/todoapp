## Version name explanation
- **0.xx.YMMDD.zz**
- **.xx** - Major improvements and changes, new important features and;
- **.YMMDD.zz** - Daily version;
- **.YMMDD** - Represents a date. *Ex.: 60210 - Feb 10th 2016*;
- **.zz** - Daily sub-version
